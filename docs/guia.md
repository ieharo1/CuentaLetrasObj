# Guia de procesamiento de texto

## Objetivo
Dominar herramientas de texto para analizar logs, archivos CSV y reportes.

## Flujo recomendado
1. Abre `index.html`.
2. Revisa los comandos con ejemplos.
3. Ejecuta los scripts en `bash/` para practicar.

## Ejemplos clave
- Contar palabras: `wc -w archivo.txt`
- Buscar patrones: `grep "error" log.txt`
- Pipeline avanzado: `cat log.txt | grep ERROR | wc -l`

## Resultado esperado
Capacidad de analizar archivos con pipelines, filtrado y transformaciones.
