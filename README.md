# Procesamiento de texto en Linux

Laboratorio para analizar texto con herramientas clasicas de Linux.

---

## Descripcion

Proyecto educativo para aprender a filtrar, transformar y contar texto usando pipelines.

### Que hace este proyecto?

- **Analisis de texto**: `wc`, `grep`, `awk`, `sed`
- **Transformaciones**: `cut`, `sort`, `uniq`, `tr`
- **Pipelines**: visualizacion paso a paso
- **Terminal simulada**: ejecucion animada
- **Scripts bash**: ejemplos completos listos para ejecutar

## Instalacion y uso

### Requisitos

- Navegador moderno
- Linux o WSL para scripts bash

### Uso

```bash
# Abrir el sitio
start index.html

# Ejecutar scripts
bash bash/word_counter.sh
bash bash/log_analyzer.sh
bash bash/text_cleaner.sh
```
